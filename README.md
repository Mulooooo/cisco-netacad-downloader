# Cisco NetAcad Downloader

Ce Userscript ajoute un bouton de téléchargement directement dans l'interface des cours Cisco NetAcad. Il permet d'exporter le contenu de la page active en un fichier **Markdown (.md)** propre et structuré (sans les menus, quiz ou bruits visuels), idéal pour résumer les cours.
